SKILL

Soft Skill

Having good verbal and written communication skills. You can easily converse with others and express ideas clearly.

Being able to work well in a team. You listen to others' opinions, contribute to discussions, and support common goals.

Having the ability to identify problems, analyze the factors involved, and find effective solutions.

Hard Skill

Proficiency in programming languages such as Python and Java.

Skills in web development using technologies such as HTML, CSS, and JavaScript.

Understanding of database design, SQL language, and database management using database management systems such as MySQL.
